# enzu.ru

This repo is for my minimalist retro homepage: https://enzu.ru

This is an example of how to use Express to host a simple static website with some niceties like automatic redirect to HTTPS.

This site has very valid HTML, CSS, and JS. It has a good [Lighthouse](https://developers.google.com/web/tools/lighthouse/) score and renders nicely in all mobile, desktop, and terminal browsers.

I always try to use the latest versions of Node and Express because this website is so simple.
